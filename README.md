Software de monitoramento de etileno desenvolvido para reduzir o desperdício de frutas em hipermercados. Através da leitura de sensores de gás, a plataforma entrega dados precisos sobre o amadurecimento das maçãs Gala, permitindo que a gestão tome decisões estratégicas para evitar descartes desnecessários.
